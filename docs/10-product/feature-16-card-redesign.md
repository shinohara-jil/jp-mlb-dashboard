# ⑯ カードデザインのリデザイン（スタッツテーブル型）

選手カードの見た目を刷新した記録。実装日 2026-06-05。

## 背景・目的

- 旧デザインは「AIが作った感」が強い、という課題があった。
- AI感の原因になりやすい要素：青のグラデ／上辺の青い太線／強い影／角丸大きめ／数字の縦並び（値の上にラベル）。
- 目的：もっと自然で、スポーツサイトらしく、数字が読みやすいデザインへ。

## 進め方（見本→選択→反映）

- 本番（index.html）には触らず、**見本専用ページ `card-designs.html`** を新規作成。
- 方向性の違う**10案**を、打者（大谷風）・投手（山本風）のサンプルでセット表示。
  1. ミニマル・ライン／2. スポーツ新聞風／3. MLB公式風／4. ボックススコア／5. ヒーロー写真／
  6. ダークモード／7. 横長リスト／8. フラット・ペール／9. マガジン／10. スタッツテーブル
- ユーザーが**案10「スタッツテーブル」**を選択 → 本番へ反映。

## 採用デザイン（案10）の要点

- 白基調＋細い罫線（`border:1px solid #d8dde3` / `border-radius:6px`）、影なし。
- **シーズン通算を表組み**で表示：上段＝項目見出し（薄グレー背景）、下段＝数値。
- 名前の下にチーム略称を小さく配置。最新試合は上部の一行に集約（「最新登板 6/1 …」）。
- **🔥（活躍）の選手だけ**、カード上辺に細いオレンジ線（`.card.hot { border-top:3px solid var(--hot) }`）。
- 数値は等幅数字（`font-variant-numeric: tabular-nums`）で桁を揃える。
- 「推移を見る →」はカード下部の帯（上罫線つき）。絵文字 📈 は外した。

  - 追加：`statTable(pairs)`（見出し行＋数値行のテーブルを生成）、`cardHead(p)`（写真＋名前＋チーム）。

  - `trendButton` の文言を「推移を見る →」に変更（📈 削除）。
  - 未使用化した `statCell()` を削除、`teamTag()` は `cardHead()` に統合。
- `index.html`：キャッシュ対策の `?v=` を `2026-06-05-redesign` に更新（style.css・app.js 両方）。
