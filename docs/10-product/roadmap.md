# 改修計画（進化ロードマップ）

日本人メジャーリーガー成績ダッシュボードを、これからどう進化させていくかの計画メモ。
2026-06-02 時点の検討内容。難易度はあくまで目安（★が多いほど大変）。

## 前提（今の作り）

- GitHub Pages で公開している静的サイト。リポジトリはパブリック。
- データ取得はブラウザ側（app.js）が MLB 公式 API から直接行う。
- 毎朝 日本時間10時ごろ、GitHub Actions が `fetch_stats.py` を実行して `data/` をコミット。
- 画面は `data/latest.json` を読んで即表示。「🔄 更新」ボタンは直接 MLB からライブ取得。

## それぞれの詳細

### ① 画像も入れたい ★☆☆ ✅ 実装済み・公開済み（2026-06-02）
- 選手の顔写真を、カード左（丸）とハイライト欄に表示。外部URL方式（追加ファイルなし）。
- 詳細・実装メモは [feature-01-images.md](feature-01-images.md) を参照。

### ② メールで届くようにしたい ★★☆
- すでに「毎朝自動で成績を取りに行く」処理が動いているので、そこに「メールを送る」ステップを追加するだけ。
- 必要なもの：メール送信サービスの登録（無料枠あり）と、送り先のアドレス。
- 注意点：リポジトリは公開なので、メールのパスワード等の秘密情報は GitHub Secrets に隠して使う。

### ③ 更新タイミングを賢く ★★☆
- 簡単版：今は朝1回だけ → 1日に数回（朝・夕・夜など）動かす。設定の数字を変えるだけ。
- こだわり版：「その日の全試合が終わったら更新」。試合終了を判定する仕組みが要るので少し工夫が必要。
- まずは簡単版で「1日3回」くらいから始めるのが現実的。

### ④ 予想を入れたい ★★☆〜★★★
- 次の登板予定日・予告先発：MLB が公式に発表しているデータがあるので比較的かんたん（★★☆）。
- 次の試合の成績予想：本格的な「予測」。当たるかは別問題で、作り込むと難しい（★★★）。
  簡単にやるなら「シーズン平均からこのくらいでは？」という目安表示にとどめる。
- おすすめは「次の登板日・予告先発」から。

### ⑤ スマホアプリにしたい ★☆☆〜★★★
- かんたん版（PWA）：今のサイトを「スマホのホーム画面に置けるアプリ風」にする。申請不要・追加費用なし（★☆☆）。
- 本格版（App Store / Google Play）：開発者登録（Appleは年間約1.5万円）・審査・作り直しが必要で大変（★★★）。
- 個人で楽しむなら PWA で十分。まずこちらがおすすめ。

### ⑥ リーグでソート（並べ替え）したい ★★☆
- MLB は「ア・リーグ（American League）」と「ナ・リーグ（National League）」の2リーグに分かれている。
  選手をリーグ別に並べ替え・絞り込みできるようにしたい。
- 必要なこと：各選手に「どちらのリーグか」の情報を持たせる。
  - 今の保存データにはチーム略称しか無いので、毎朝の取得（fetch_stats.py）で
    チーム→リーグの対応を引いて、各選手に `league`（AL／NL）を1項目足す軽い改修が必要。
  - 画面側（app.js）に「リーグで並べ替え／絞り込み」のボタンや見出しを追加する。
- 難易度は★★☆。データに1項目足す＋画面に切り替えUIを足す、という構成。
- 補足：表示方法は「投手・野手の中をさらにリーグ順に並べる」「リーグで絞り込むボタン」「リーグごとに見出しで区切る」などの選択肢がある（着手時に詰める）。

### ⑦ 試合中のデータに「試合中」タグを付けたい ★★☆
- 今は「最新試合」として直近1試合の成績を出しているが、その試合が**まだ進行中**の場合、
  表示している数字は「途中経過」になる。それを見分けられるよう「🟢 試合中」などの目印を付けたい。
- 必要なこと：その試合が「進行中か／終了済みか」を判定する。
  - MLB のデータには試合の状態（`Preview`＝開始前／`Live`＝進行中／`Final`＝終了）がある。
    今すでに使っている試合情報の取得（schedule）から、この状態も一緒に読めば判定できる。
  - 進行中なら、カードやハイライトに「試合中」タグを表示する（成績は途中経過の注記つき）。
- 難易度は★★☆。判定用の状態を1つ読み取り、画面にタグを足す構成。
- 補足：fetch_stats.py（毎朝の取得）とブラウザ側のライブ取得（🔄更新）の両方に同じ判定を入れると、
  どちらの経路でも「試合中」が正しく出る。着手時に詰める。

## おすすめの進め方（やりやすい順）

1. ① 画像（すぐ見栄えが良くなる・簡単）← **まずここから着手**
2. ⑤ PWA化（スマホで使いやすく・簡単）
3. ② メール通知 ＋ ③ 1日複数回更新（セットで相性が良い）
4. ④ 次の登板日表示（データがあるので現実的）
5. こだわり版（全試合終了判定／成績予想）は、上が落ち着いてから
