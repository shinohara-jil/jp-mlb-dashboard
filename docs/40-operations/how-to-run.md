# 使い方（起動方法）

## 必要なもの

- Python（3.x）がインストールされていること。**それ以外の追加インストールは不要**。
  - このツールはPython標準機能だけで動くので、`pip install` などは必要ありません。

## 起動のしかた

1. このフォルダ（`260601`）を開く。
2. `app.py` を実行する。
   - 方法A：ファイルをダブルクリック（環境によってはコマンド画面が一瞬で閉じることがある）。
   - 方法B（おすすめ）：このフォルダでコマンドを開いて `python app.py` と入力。
3. 自動でブラウザが開き、ダッシュボードが表示されます。
   - 開かない場合は、ブラウザのアドレス欄に `http://127.0.0.1:8000/` と入力。
4. 終了するときは、コマンド画面で `Ctrl + C` を押す。

## 使い方

- 画面右上の **「🔄 更新」ボタン** を押すと、MLB公式サービスから最新の成績を取り直して表示します（数秒）。
- 一度更新すると、その時点のデータが `data/latest.json` に保存され、次に開いたときはすぐ表示されます。
- 日付ごとのスナップショットが `data/history/` に溜まっていきます（あとで振り返り用）。

## 選手の日本語名を増やす・直すには

- `config/name_map.json` を編集します（英語名 → 日本語名）。
- ここに無い選手は英語名のまま表示されます。成績の取得には影響しません。

## うまくいかないとき

- **画面が真っ白／データが出ない**：右上の「🔄 更新」を一度押してください（初回はデータ取得が必要）。
- **「取得に失敗しました」と出る**：MLB公式サービスが一時的に混んでいる可能性。少し待って再度「更新」。
- **ポート8000が使用中というエラー**：他のアプリが同じ番号を使っています。`app.py` の `PORT = 8000` を `8001` など別の番号に変えてください。
